Constructors

UIColorPicker (namespace, parent)
Creates new UIColorPicker with its elements

Parameters:

  • table namespace — Your mod namespace
  • ScriptUI/GalaxyMap/UIContainer parent — Parent element

    (!) If you're planning to use color picker on Galaxy Map, pass GalaxyMap() as parent, NOT its child elements

Returns:

    table — UIColorPicker instance

Usage:

    local uiColorPicker
function MyModNamespace.initUI()
    uiColorPicker = UIColorPicker(MyModNamespace, ScriptUI())
end

UIColorPicker: Callbacks

onApplyCallback (color, element)
Fires when user presses 'Apply' button if it was specified in the 'show' call If function will return 'false', the color picker window won't close

Parameters:

  • Color color — Color that user selected
  • UIColorPicker element — This color picker

Usage:

    function MyModNamespace.onColorPickerApply(color, element)
    if someCheckPassed(color) then
        -- do stuff
    else -- check failed, don't hide the window
        return false
    end
end
onCancelCallback (element)
Fires when user presses 'Cancel' button if it was specified in the 'show' call If function will return 'false', the color picker window won't close

Parameters:

  • UIColorPicker element — This color picker

UIColorPicker: Methods

hide ()
Hides color picker
setAlpha (value)
Sets alpha value. If selected mode doesn't have alpha, this function will do nothing

Parameters:

  • float value — Alpha value from 0 to 1
setHue (value)
Sets color picker hue

Parameters:

  • float value — Hue value from 0 to 360
setSaturation (value)
Sets saturation. If selected mode doesn't have saturation, this function will do nothing

Parameters:

  • float value — Saturation value from 0 to 1
setValue (value)
Sets color value. If selected mode doesn't have value, this function will do nothing

Parameters:

  • float value — Color value from 0 to 1
show ([size=vec2(400，300)[, title="Pick a color"%_t[, mode="HSVA"[, defaultColor=ColorRGB(1，0，0)[, onApplyCallback[, onCancelCallback[, valueSliderMin=0[, valueSliderMax=1[, alphaSliderMin=0[, alphaSliderMax=1]]]]]]]]]])
Shows color picker window, adjusted to desired mode and size

Parameters:

  • vec2 size (default: vec2(400，300)) — Window size
  • string title (default: "Pick a color"%_t) — Window caption
  • string mode (default: "HSVA") — Color mode. Can be: HS, HSA, HSV, HSVA, HV, HVA, HVS, HVSA
  • Color defaultColor (default: ColorRGB(1，0，0)) — Color that will be set in color picker
  • string onApplyCallback (optional) — Callback that will fire when user will press 'Apply' button
  • string onCancelCallback (optional) — Callback that will fire when user will press 'Cancel' button
  • float valueSliderMin (default: 0) — Minimal possible color value slider value
  • float valueSliderMax (default: 1) — Maximal possible color value slider value
  • float alphaSliderMin (default: 0) — Minimal possible alpha slider value
  • float alphaSliderMax (default: 1) — Maximal possible alpha slider value

See also:

  • UIColorPicker:onApplyCallback
  • UIColorPicker:onCancelCallback

Usage:

    function MyModNamespace.onShowColorPickerBtnPressed()
    uiColorPicker:show(vec2(400, 300), "Pick a color"%_t, "HSVA", ColorRGB(1, 0, 0), "onColorPickerApply", "onColorPickerCancel", 0.5, 1)
end
update (timeStep)
Updates color picker (checks for mouse position and clicks). This function should run every tick

Parameters:

  • float timeStep — Time step variable from update or updateClient function

Usage:

    function MyModNamespace.getUpdateInterval()
    if colorPicker and colorPicker.visible then return 0 end -- every tick
    return 0.5
end
function MyModNamespace.updateClient(timeStep)
    colorPicker:update(timeStep)
end
